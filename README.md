#  Weather App

Una aplicación del clima moderna, profesional y altamente interactiva construida con React, TypeScript y TailwindCSS. Ofrece información meteorológica en tiempo real con una interfaz visual impresionante que se adapta dinámicamente a las condiciones climáticas.

##  Características

###  Interfaz Dinámica
- **Fondos animados** que cambian según las condiciones climáticas
- **Transiciones suaves** con Framer Motion
- **Diseño glassmorphism** con efectos de cristal
- **Modo día/noche** automático

###  Información Meteorológica
- **Clima actual** con temperatura, sensación térmica, humedad y viento
- **Pronóstico por horas** (próximas 24 horas)
- **Pronóstico de 5 días** con máximas y mínimas
- **Detalles completos**: presión, visibilidad, nubes, amanecer/atardecer

###  Búsqueda Inteligente
- **Búsqueda de ciudades** con autocompletado
- **Geolocalización** automática
- **Historial de búsquedas** reciente
- **Ciudades favoritas** guardadas localmente

###  Personalización
- **Unidades de temperatura**: Celsius / Fahrenheit
- **Idioma**: Español / English
- **Actualización automática** cada 5 minutos

##  Tecnologías

- React 18 + TypeScript 5
- Vite 7
- TailwindCSS 4
- Framer Motion
- Zustand (Estado Global)
- TanStack Query
- Axios
- date-fns
- Lucide React
- OpenWeatherMap API

##  Instalación

```bash
git clone https://github.com/tu-usuario/weather-app.git
cd weather-app
npm install
npm run dev
```
